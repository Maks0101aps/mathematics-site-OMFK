# 3D Геометрія

Інтерактивний освітній вебдодаток для уроків математики та геометрії. Проєкт допомагає учням вивчати 3D-фігури через візуалізацію, обертання моделей, зміну параметрів, автоматичні обчислення та порівняння з об'єктами з реального життя.

## Можливості

- інтерактивна 3D-сцена з обертанням і масштабуванням;
- фігури: куб, паралелепіпед, куля, циліндр, конус, піраміда, призма;
- налаштування параметрів кожної фігури в реальному часі;
- автоматичний розрахунок об'єму та площі поверхні;
- красиве відображення формул через KaTeX;
- режим прикладів із реального життя;
- навчальні підказки та міні-завдання;
- адаптивний інтерфейс для ноутбуків, планшетів і телефонів.

## Технології

## Встановлення

Клонуйте репозиторій:

```bash
git clone <URL_РЕПОЗИТОРІЮ>
cd mathematics-site-OMFK
```

Встановіть усі залежності:

```bash
npm run install:all
```

Ця команда встановлює npm-залежності та браузер Chromium для локальної перевірки через Playwright.

## Запуск у режимі розробки

```bash
npm run dev
```

Після запуску відкрийте сайт у браузері:

```text
http://localhost:3000
```

## Збірка проєкту

```bash
npm run build
```

## Доступні команди

```bash
npm run install:all
npm run dev
npm run build
```

## Структура проєкту

```text
src/app/page.tsx                  головна сторінка та інтерфейс
src/components/geometry-scene.tsx  3D-сцена та моделі фігур
src/lib/figures.ts                 дані фігур, формули та обчислення
src/app/globals.css                глобальні стилі
```
